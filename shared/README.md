# Shared

This folder is reserved for code shared between the frontend and backend.

Planned contents (future phases):

- **Types** — Shared TypeScript/Python type definitions where applicable
- **Constants** — Application-wide constants (e.g. subject names, enums)
- **Schemas** — Shared validation schemas or API contract definitions
- **Utilities** — Cross-cutting helpers used by both services

No implementation in Phase 0. Add modules here when a concrete shared need arises in later phases.
